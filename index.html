Movie Recommendation Project
Python • Machine Learning

A machine learning–based recommendation engine that utilizes collaborative filtering to suggest movies based on user preferences. Users interact with the system by swiping left or right on movie cards, indicating dislikes and likes respectively. The model is trained on two primary datasets—movies and user ratings—and currently achieves an impressive RMSE of 0.87 and MAE of 0.67.

The project features a dynamic frontend interface that visually presents the movie options and captures user feedback in real time. While the core model is fully functional, frontend development is ongoing to enhance user experience. Meanwhile, model tuning and optimization continue to ensure personalized, accurate recommendations for each user.
